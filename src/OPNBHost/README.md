# FPC Server (minimale)

Progetto Free Pascal minimale per test e sviluppo.

## Build

Apri `cmd.exe` nella cartella `src/fpc_server`.
Compila con il compilatore Free Pascal (`fpc`):

```
fpc server.lpr
```

Questo produce `server.exe` su Windows.

## Run

Esegui:

```
server.exe
```

Premere Invio per terminare il programma.
